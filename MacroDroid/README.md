# MacroDroid Folder

Copy the entire contents of this folder (including the `.macro` file and the `assets/` subfolder) to the `/sdcard/MacroDroid/` directory on your Android device.

After copying, open MacroDroid and import the `Recuperar_conexión.macro` file.

**Prerequisites:**

- Install MacroDroid from the Play Store.
- Grant the necessary permissions via ADB hack or Shizuku (see main repository for details).

For full instructions, visit: https://github.com/rodnye/airplane-mode.macrodroid
